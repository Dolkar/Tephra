tp::PipelineCache class

Speeds up the compilation of pipelines by allowing the result of pipeline compilation to be reused between pipelines and between application runs.

Contents

  • Reference
    • Public functions

Remark

Access to the tp::PipelineCache object is internally synchronized, meaning it is safe to operate on it from multiple threads at the same time.
